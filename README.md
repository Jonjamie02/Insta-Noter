# Insta-Noter

Insta-Noter is a self-hosted web application to view, create, and delete Instagram Notes using the [`instagram-notes`](https://github.com/xydv/instagram-notes) Node.js package.

---

## Features

* View current Instagram Notes
* Create new Notes (1â€“60 characters)
* Delete existing Notes
* Self-hosted and easy to set up
* Your credentials stay on your server

---

## Prerequisites

* Node.js (v14 or higher recommended)
* npm (comes with Node.js)
* An Instagram account

---

## Getting Started (Local Development)

### 1. Clone the repository

```bash
git clone https://github.com/ashyrv0/Insta-Noter.git
cd Insta-Noter/backend
```

### 2. Install dependencies

```bash
npm install
```

### 3. Start the server

```bash
npm start
```

You should see:

```
Server running on port 3000
```

### 4. Access the application

Open your browser and go to:

```
http://localhost:3000
```

### 5. Login

1. Enter your Instagram username
2. Enter your Instagram password
3. Click "Login"

Once logged in, you can view, create, and delete your Instagram Notes.
